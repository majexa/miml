Обзор

Главным нововведением в MiMl2 является компонент навигации для списокв. Он подходит для тех случаев когда просмотр содержимого страниц хотелось бы делать моментально без загрузки контента с сервера.

Это может быть всё, что угодно: список произведений для прочтения, список тем форума, список новостей. Все те случаи, когда хотелось бы быстренько посмотреть содержимое страницы, а ждать ответа от сервера слишком долгая задача особенно для мобильного интернета.

API

Функионал списков настраивается опциями класса Ngn.FramesSlider.List

  • titleSelector String значение по умолчанию: .header .title
    CSS селектор для заголовка страницы
  • listSelector String значение по умолчанию: .list
    CSS селектор для контейнера со списком
  • listItemSelector String значение по умолчанию: .item
    CSS селектор элемента списка
  • itemTitleSelector String значение по умолчанию: .title
    CSS селектор для заголовка элемента списка
  • itemTitleSelector Array of Strings значение по умолчанию: ['.text']
    CSS селекторы для элементов, которые будут отображаться во втором фрейме

Пример 1. Базовый

Демо на сайте Демо для мибилки
Список из заголовка и текста, который открывается на втором слайде. В HTML-коде использован лейаут с теми классами, что указаны в значениях по умолчанию (см. API).

JS

      
new Ngn.ListSlider(
  new Ngn.FramesSlider(
    document.getElement('#framesBody')
  )
);

HTML

    <div id="framesBody">
  <div class="frame">
    <div class="list">
      <div class="item">
        <div class="title">title</div>
        <div class="text">html for second slide</div>
      </div>
    </div>
  </div>
</div>
